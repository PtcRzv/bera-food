# Bera Food Website

## Overview
Bera Food is a fast food restaurant located in Timișoara, specializing in authentic shawarma and a variety of fast food options. This project serves as the official website for Bera Food, providing users with information about the menu, gallery, reviews, and contact details.

## Project Structure
The project consists of the following files:

- **index.html**: The main HTML document that structures the webpage.
- **styles.css**: The stylesheet that defines the visual appearance of the website.
- **script.js**: The JavaScript file that adds interactivity to the website.
- **.gitignore**: A file that specifies which files and directories should be ignored by Git.
- **README.md**: This documentation file.

## Installation
To run the Bera Food website locally, follow these steps:

1. Clone the repository:
   ```
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```
   cd bera-food
   ```
3. Open `index.html` in your web browser to view the website.

## Usage
- The website features a hero section that highlights the restaurant's offerings.
- Users can navigate through the menu, view the gallery, read reviews, and find contact information.
- The website is responsive and works well on both desktop and mobile devices.
